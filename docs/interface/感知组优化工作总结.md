## 🔧 核心优化详情

### 1. 卡尔曼滤波优化

#### KalmanCV2DAdaptive (4状态)
```python
# 新增特性：
- 自适应过程噪声：根据速度自动调整 Q 矩阵
- 自适应观测噪声：根据检测置信度调整 R 矩阵
- 创新异常检测：检测遮挡或误检
- 速度置信度计算：评估速度估计的可靠性
```

#### KalmanBoTAdaptive (8状态)
```python
# 新增特性：
- 运动模式检测：stationary / slow / fast
- 加速度敏感噪声：检测速度变化并调整
- 增强轨迹预测：支持不确定性传播
```

### 2. 轨迹预测

#### TrajectoryPredictor
```python
# 功能：
- 多步未来预测 (默认10步)
- 置信度衰减建模
- 不确定性椭圆计算
```

#### TrajectorySmoother
```python
# 功能：
- RTS 平滑算法
- 加权轨迹平滑
- 速度一致性保持
```

#### TrajectoryAnalyzer
```python
# 功能：
- 运动模式分类
- 曲率计算
- 方向变化检测
- 出框时间预估
```

#### IdentityManager

## 📡 ROS2 接口增强

### TargetTrack.msg (调度组对接)

```msg
uint32 target_id              # 原有
float64 x, y                 # 原有
float64 vx, vy               # 原有

# 新增调度字段
float32 confidence            # 检测置信度
uint8 cls                     # 目标类别
bool is_confirmed            # 是否确认
float32 speed                # 速度大小
uint8 motion_mode            # 运动模式 (0-3)
float32[5] pred_x           # 预测X (5步)
float32[5] pred_y           # 预测Y (5步)
float32[5] pred_conf        # 预测置信度
```

### EnclosureTarget.msg (封控组对接)

```msg
uint32 target_id
float64 x, y
float32 speed
uint8 motion_mode
float32 confidence

# 包围盒信息
float32 box_x1, box_y1, box_x2, box_y2

# 预测轨迹
float32[5] pred_x, pred_y

# 历史轨迹
float32[10] history_x, history_y
```

### EnclosureTargetArray.msg

```msg
std_msgs/Header header
uint32 frame_idx
EnclosureTarget[] targets

# 无人机位置 (多机编队参考)
float32[8] drone_x, drone_y
uint8 num_drones

# 围控参数
float32 enclosure_radius
float32 min_enclosure_dist
```

## 🚀 使用方式

### 1. 启动优化版跟踪器

```bash
# 使用优化配置
ros2 run perception_pkg tracker_node \
    --ros-args \
    -p tracker.kind:=botsort_adaptive \
    -p enclosure.enabled:=true \
    -p enclosure.topic:=/enclosure_targets
```

### 2. 订阅目标数据

```bash
# 调度组订阅
ros2 topic echo /target_track

# 封控组订阅
ros2 topic echo /enclosure_targets
```

### 3. 运行测试
